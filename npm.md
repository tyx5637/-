# npm

1. npm可以用来
  
   - 为您的应用调整代码包，或者按照原样合并包
   - 下载你可以立即使用的独立工具
   - 使用npx运行包而不下载
   - 随时随地与任何使用npm的用户分享代码
   - 将代码限制为特定人员
   - 创建组织来协调包维护、编码和开发人员。
   - 通过使用组织形成虚拟团队。
   - 管理多个版本的代码和代码依赖项。
   - 更新底层代码时轻松更新应用程序。
   - 发现解决同一个难题的多种方法。
   - 查找正在处理类似问题和项目的其他开发人员。
2. npm login —— 创建新账号
3. npm whoami —— 检测是否成功创建，创建成功的话会打印用户名
